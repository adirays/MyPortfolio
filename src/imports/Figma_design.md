# Personal Portfolio Website Brief

## Experience structure
### Page 1: Landing
- Full-screen minimal intro
- Large stylized name centered or slightly off-center
- One short descriptor line beneath
- One CTA button to enter portfolio
- Subtle background animation or atmospheric effect
- No navigation, no extra clutter

### Page 2: Portfolio
- Animated transition from landing into portfolio
- Modular sections with strong spacing
- Interactive project cards
- Smooth scroll-based reveals
- Hover depth and motion
- Premium visual feel throughout

## Content sections
1. Hero / Intro
2. Featured Projects
3. Case Studies
4. Skills / Tech Stack
5. Experience / Timeline
6. Contact
7. Optional Now Page

## Visual style
- Dark, premium, minimal, futuristic
- Large typography
- Clean grid
- Soft glows, blur, gradient accents
- Editorial spacing
- Refined hover states
- Cinematic motion

## Interaction style
- Seamless page transition
- Smooth reveal animations
- Hover tooltips or image previews for projects
- Subtle cursor interactions
- Copy-to-clipboard email action
- Gentle scroll progress indicator

## Must-haves
- Mobile responsive
- Fast loading
- Strong typography
- Clear CTA
- Personal brand identity
- Designed for recruiters and creative tech audiences
